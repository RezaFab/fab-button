# @rezafab/fab-button-styles

Default CSS styles for FabButton.

## Install

```bash
pnpm add @rezafab/fab-button-styles
```

## Usage

```css
@import "@rezafab/fab-button-styles/style.css";
```

## Recommendation

For most projects, install `@rezafab/fab-button` and import:

```css
@import "@rezafab/fab-button/styles.css";
```

Use this package directly only if you want CSS-only consumption with explicit package control.

## Why this package exists

- Centralizes default FabButton styles in one package.
- Keeps styling reusable across all framework adapters.
- Allows advanced users to consume styles without pulling adapter code.

Main docs: https://github.com/RezaFab/fab-button#readme
